Как установить — один раз

  1. Нажми жёлтую кнопку «Скачать Tempora».
  2. Внизу появится файл — нажми на него.
  3. Если телефон спросит про «неизвестный источник» — нажми «Настройки», включи «Разрешить из этого источника», вернись назад.
  4. Нажми «Установить». Готово!
Дальше Tempora обновляется сама по интернету — этот шаг больше никогда не нужен.
На iPhone Tempora ставится из App Store — пришлю ссылку, когда Apple одобрит.
