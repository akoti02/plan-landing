Tempora

Чистая установка — свежая версия без «Режима разработчика».

СНАЧАЛА удали старую версию (обязательно):
  1. Найди значок Tempora → задержи палец → «Удалить приложение» → «Удалить».
  2. Проверь, что значка нет ни на экране, ни в «Библиотеке приложений».
  3. Перезагрузи iPhone (выключи и включи) — это сбрасывает старую установку.
  4. Только теперь нажимай кнопку ниже.
Установить заново
Дальше:
  1. iPhone спросит «Установить “Tempora”?» — нажми Установить.
  2. Закрой Safari, найди значок Tempora — он будет ставиться.
  3. Первый запуск пишет «Ненадёжный разработчик»? Это нормально: Настройки → Основные → VPN и управление устройством → выбери разработчика → Доверять.
  4. Открывай Tempora. «Режима разработчика» быть не должно.

Открой эту страницу в Safari (не Chrome, не внутри другого приложения), иначе кнопка не сработает.
