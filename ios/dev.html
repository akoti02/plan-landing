Сборка для разработки: после установки правки UI прилетают
сами по Wi-Fi (нужен запущенный dev-сервер на ПК).

УстанавливаетсяBUILD 106нажми, чтобы установить на iPhone →
Как поставить:
  1. Один раз включи Режим разработчика: Настройки → Конфиденциальность и безопасность → Режим разработчика → вкл → перезагрузка (для dev-сборок Apple требует это всегда).
  2. Нажми большую зелёную кнопку выше → Установить.
  3. Если «Ненадёжный разработчик»: Настройки → Основные → VPN и управление устройством → выбрать разработчика → Доверять.
  4. Запусти — приложение само уйдёт на dev.akoev.com и будет перезагружаться при каждой правке.

Открывать в Safari. Если экран пустой/«не удалось открыть страницу» — значит на ПК не поднят dev-сервер (esbuild + туннель).
